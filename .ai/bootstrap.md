# AI Agent Bootstrap

Purpose: minimal startup procedure for any AI agent working in this repository.

## Startup Procedure
1) Read [.ai/knowledge/project-context.md](./knowledge/project-context.md).
2) Read [.spec/constitution.md](../.spec/constitution.md).
3) Read [.spec/index.md](../.spec/index.md).
4) Read [docs/core/DOCS_INDEX.md](../docs/core/DOCS_INDEX.md).
5) Read [docs/core/SYSTEM_MAP.md](../docs/core/SYSTEM_MAP.md).
6) Read [.ai/orchestrator.md](./orchestrator.md) for agent flow and handoffs.
7) Read the active spec under [.spec/specs/](../.spec/specs/) if a task is in progress.
8) Verify repository state with git status -sb.
9) Inspect relevant code and docs before proposing changes.

## Non-Negotiable Constraints
- Do not modify runtime code unless the active spec and plan explicitly require it.
- Do not violate [docs/core/ARCHITECTURE_CONTEXT.md](../docs/core/ARCHITECTURE_CONTEXT.md).
- Do not change build files or move documentation unless explicitly scoped.
- Keep changes small, reviewable, and reversible.
